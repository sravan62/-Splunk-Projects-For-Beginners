# -Splunk-Projects-For-Beginners
A collection of beginner-friendly projects focused on learning and practicing log analysis using Splunk. This repository covers real-world SOC use cases such as DNS log analysis, network traffic monitoring, detecting suspicious activities, and basic threat hunting techniques.

## Projects

[DNS Log Analysis with Splunk](https://github.com/sravan62/-Splunk-Projects-For-Beginners/blob/main/DNS%20Log%20Analysis%20with%20Splunk.md) : This project demonstrates a structured approach to analyzing DNS (Domain Name System) logs using Splunk SIEM. It includes steps for ingesting log data, extracting key fields, and performing analysis to identify DNS query trends. The project also focuses on detecting suspicious activities, uncovering anomalies, and monitoring DNS traffic to support effective threat detection.
